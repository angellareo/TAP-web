How to enter data
Test Information
  • # Students — total number of examinees; must match the rows you paste below.
  • # Test Items — number of questions on the test.
  • Offset — characters to skip at the start of each row (e.g. a student ID of length 5 → offset 5). Leave 0 if responses start at column 1.
Items Configuration
  • Answer Key String — type or paste the full string of correct answers (e.g. ACBAABAABC). The table below will reflect each character.
  • # Options — number of answer choices for each item (2–5).
  • Include — uncheck to exclude an item from analysis (e.g. a faulty question).
Student Response Data
  • Paste one row per student. Each character should correspond to the option chosen (e.g. A, B, or a digit like 1, 2).
  • Characters must use the same coding as the answer key.
  • Rows are colour-coded below the text area: green = correct length, red = wrong length.
Actions
  • Save .tap file — download the current data as a reusable .tap file you can load next time.
  • Analyse — run the full test analysis and view results.
